TaskFlow 🗂️

TaskFlow é um gerenciador de notas em terminal (CLI) simples, feito para organizar tarefas e anotações de forma rápida, leve e eficiente diretamente pelo terminal.

🚀 Funcionalidades
Criar notas rapidamente pelo terminal
Listar todas as notas salvas
Marcar notas como concluídas
Remover notas
Armazenamento local simples (sem dependências externas)
Interface minimalista focada em produtividade
